Acerca de Biblioteca IBG

Sobre Nosotros

Biblioteca IBG es una aplicación de gestión de biblioteca desarrollada para el Instituto Bíblico Global. Nuestro objetivo es proporcionar una plataforma moderna y eficiente para gestionar un catálogo extenso de libros relacionados con temáticas bíblicas, teológicas y de crecimiento cristiano.

Características Principales

  • Catálogo Completo: Más de 600 títulos disponibles
  • Sistema de Búsqueda Avanzado: Filtra libros por título, autor y categoría
  • Gestión de Préstamos: Solicita y controla tus préstamos fácilmente
  • Autenticación Segura: Crea tu cuenta y accede a tus datos protegidos
  • Interfaz Intuitiva: Diseño responsivo con Bootstrap
Estadísticas

Sistema desarrollado con:

  • Django 6.0
  • Python 3.11+
  • PostgreSQL/SQLite
  • Bootstrap 5.3

Tecnologías y Competencias Implementadas

1. Framework Django

Utilizamos Django como nuestro framework principal para desarrollar una aplicación web escalable y segura. Django nos proporciona herramientas de administración integradas, ORM potente y seguridad robusta contra ataques comunes como CSRF, SQL Injection y XSS.

2. Modelos de Base de Datos

Implementamos modelos para:

  • Autor: Gestión de autores de libros
  • Categoría: Clasificación de libros por tipo
  • Libro: Información completa de libros con relaciones
  • Préstamo: Control de préstamos y devoluciones
3. Autenticación y Autorización

Sistema de autenticación de Django con:

  • Registro de nuevos usuarios
  • Login seguro con sesiones
  • Control de acceso basado en permisos
  • Roles de administrador y usuario regular
4. Formularios Django

Implementamos formularios personalizados para:

  • Búsqueda y filtrado de libros
  • Registro de usuarios
  • Solicitud de préstamos
  • Actualización de perfiles
5. Panel de Administración

Configuramos y personalizamos el panel de administración de Django con:

  • Gestión completa de libros, autores y categorías
  • Control de préstamos y devoluciones
  • Búsqueda y filtrado avanzado
  • Auditoría de cambios
6. Templates y Frontend

Desarrollamos una interfaz intuitiva con Bootstrap 5 que incluye:

  • Diseño responsivo y moderno
  • Navegación clara y accesible
  • Formularios validados y con feedback
  • Mensajes de usuario dinámicos
Proyecto de Evaluación
